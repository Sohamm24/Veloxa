## Cycle Enthusiasts Application

Veloxa is a web application designed for cycle enthusiasts, built using the MERN stack and featuring Leaflet API for mapping and Firebase Authentication.

## Features
- **Interactive Maps**: Utilizes Leaflet API to display cycling routes and points of interest.
- **Route Planning**: Plan custom routes with ease, taking into account distance and elevation.
- **Weather Integration**: Real-time weather updates along your planned route (future enhancement).
- **User Authentication**: Secure user accounts using Firebase Authentication.
- **Profile Management**: Manage your cycling profile and track your activities.
- **Social Features**: Share routes, join groups, and connect with other cyclists (future enhancement).

## Technologies Used
- **Frontend**: React
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Mapping**: Leaflet API
- **Authentication**: Firebase Auth
- **Version Control**: Git
